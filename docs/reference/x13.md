# X-13ARIMA-SEATS (x13)

!!! warning "Not implemented in this port"
    X-13ARIMA-SEATS seasonal adjustment is **not** part of TimeSeriesEcon.m. The
    page exists so the navigation matches the sister
    [Python](https://nic2020.github.io/TimeSeriesEconPy/) and Julia
    documentation.

In `TimeSeriesEcon.jl` the `x13` module is a wrapper around the external
X-13ARIMA-SEATS program (US Census Bureau): it builds a specification, shells out
to the binary, and parses the result tables. That is a large surface (~20
specification sections and a binary dependency) and is out of scope for the
current MATLAB port, alongside the DataEcon binary file format.

If you need seasonal adjustment from MATLAB today:

- Use the X-13 binary directly via `system(...)` with a hand-written `.spc`
  file, or a community MATLAB wrapper around it; or
- Run the adjustment in Julia (`TimeSeriesEcon.jl`) or Python
  (`TimeSeriesEconPy`) and exchange the series as plain arrays.

A future milestone may add a thin `+tse/+x13` wrapper following the same staged
plan described in the repository's extension plan (a minimal `series` + `x11`
path first, then broader spec coverage). See
[`PLAN_EXTENSIONS.md`](https://github.com/jasonjensen/timeseriesecon.m/blob/main/PLAN_EXTENSIONS.md).
